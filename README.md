# vim-automkdir

Automatically call mkdir -p as necessary when writing files.

## Installation

### With [Vundle][vundle]

In vimrc:

```vim
Bundle 'benizi/vim-automkdir'
```

### With [vim-addon-manager][vam]

```vim
:InstallAddons github:benizi/vim-automkdir
```

## Usage

Open a file, even if its directory doesn't exist.

Save it.

## Full documentation

Within `vim` run:

```vim
:help automkdir
```
